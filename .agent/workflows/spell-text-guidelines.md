# Spell Card Text Guidelines

Follow these rules when adding new spell cards or editing existing ones to ensure consistency.

## Übersicht: Zwei Textfelder

| Feld | Anzeige | Inhalt |
|------|---------|--------|
| `description` | Kartenvorderseite (Hover) | **Nur Spieleffekt** - kurz & verständlich |
| `tooltip` | Rechtsklick-Modal | Philosophischer Hintergrund (wie bei Philosophen) |

---

## 1. Description (Kartenvorderseite)

*   **Inhalt:** NUR der Spieleffekt - keine Philosophie!
*   **Länge:** So kurz wie möglich, aber verständlich (~30-80 Zeichen)
*   **Ton:** Klar, technisch, eindeutig
*   **Regeln:**
    *   "Dialektik" = Mana
    *   "Glaubwürdigkeit" = Spieler-Leben
    *   "Leben" = Philosophen-Leben
    *   "Philosoph(en)" nicht "Minion(s)"

### Beispiele:

| Karte | Description |
|-------|-------------|
| Meditation | Stelle 6 Glaubwürdigkeit wieder her. |
| Cogito | Ziehe 1 Karte. |
| Aporia | Verursache 6 Schaden. |
| Axiom | +1 Dialektik in diesem Zug. |
| Tabula Rasa | Zerstöre ALLE Philosophen auf dem Feld. |
| Kontemplation | Sieh die obersten 3 Karten, nimm 1 auf die Hand. |

---

## 2. Tooltip (Rechtsklick-Detail)

*   **Länge:** Detailliert aber prägnant (~2-3 Sätze)
*   **Struktur:**
    1.  **Zitat:** (Optional) Nur wenn es ein prägnantes, bekanntes Zitat gibt
    2.  **Kernkonzept:** Ernste, akkurate Erklärung der philosophischen Idee
    3.  **Interessanter Fakt:** Ende mit **EINEM** Fakt, Anekdote oder überraschenden Detail
        *   Unbestätigte Fakten mit "angeblich" kennzeichnen
        *   **KEIN** "fun fact"
*   **Ton:**
    *   *Erster Teil:* Seriös, lehrreich
    *   *Letzter Satz:* Darf humorvoll, ironisch oder überraschend sein

### Beispiele:

| Karte | Tooltip |
|-------|---------|
| Cogito | "Cogito ergo sum" - der unerschütterliche Ausgangspunkt von Descartes' Philosophie. Selbst wenn ein böser Dämon alle Sinne täuscht, kann das denkende Ich nicht bezweifelt werden. Descartes kam auf diese Idee angeblich in einem überhitzten Zimmer während eines Winterquartiers. |
| Aporia | In der sokratischen Methode der Zustand, in dem alle Diskutanten erkennen, dass sie die Antwort nicht wissen. Sokrates führte seine Gesprächspartner absichtlich in die Aporia, um sie zum echten Nachdenken zu bringen. Manche empfanden das als so unangenehm, dass sie ihn für einen "Zitterrochen" hielten, der andere betäubt. |
| Tabula Rasa | Lockes Metapher für den Geist bei Geburt: eine "unbeschriebene Tafel" ohne angeborene Ideen. Alle Erkenntnis stammt aus Erfahrung. Die Idee wurde später von Behavioristen aufgegriffen, die glaubten, jeden Menschen zu allem erziehen zu können. |

## Schnellreferenz: Zaubertypen

| Typ | Description-Stil |
|-----|------------------|
| Schaden | "Verursache X Schaden." |
| Heilung | "Stelle X Glaubwürdigkeit/Leben wieder her." |
| Kartenziehen | "Ziehe X Karte(n)." |
| Buff | "+X/+X für einen Philosophen." |
| Mana | "+X Dialektik" / "Sperre X Dialektik des Gegners." |
| Board Clear | "Zerstöre alle..." |
| Suche | "Sieh/Suche... nimm auf die Hand." |
